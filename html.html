HTML

HTML, singkatan dari HyperText Markup Language, adalah bahasa markup standar yang digunakan untuk membuat dan menyusun halaman web di internet. HTML bukanlah bahasa pemrograman seperti JavaScript atau Python, melainkan bahasa markah (markup language) yang berfungsi untuk memberi struktur dan makna pada berbagai elemen dalam sebuah halaman web.

HTML menggunakan tag-tag tertentu untuk menunjukkan berbagai jenis konten, seperti paragraf, heading, gambar, tautan, tabel, form, dan elemen-elemen multimedia. Tag-tag ini dibuka dan ditutup untuk membungkus isi atau konten, dan biasanya dibentuk dengan tanda kurung sudut, contohnya (p) untuk paragraf, (h1) untuk judul, dan (a) untuk tautan.

Fungsi Utama HTML :

  • Membuat Struktur Halaman Web: HTML menyusun elemen-elemen pada halaman agar tersusun secara hierarkis, mulai dari header, body, hingga footer. Ini memudahkan browser untuk merender (menampilkan) halaman sesuai dengan struktur yang diinginkan oleh pembuatnya.
  • Tampilan Dasar Konten: Meskipun desain web modern lebih mengandalkan CSS, HTML tetap bertanggung jawab dalam memberikan struktur dasar seperti heading, list (daftar), tabel, dan form.
  • Menyisipkan Konten Multimedia: HTML memungkinkan penyisipan gambar (img), video (video), dan audio (audio) ke dalam halaman web, menjadikannya lebih interaktif dan menarik.
  • Mendukung Navigasi Web: Tag (a) digunakan untuk membuat hyperlink antarhalaman, baik dalam satu situs maupun ke situs lain. Ini merupakan bagian penting dari hypertext, yang menjadi dasar dari World Wide Web.
  • Bekerja Bersama CSS dan JavaScript: HTML bekerja secara sinergis dengan CSS (untuk styling atau tampilan) dan JavaScript (untuk interaktivitas dan logika), membentuk fondasi utama dari pengembangan web modern.
